Kafkaи°ғз ”пјҲ1пјүпјҡи·Ёж•°жҚ®дёӯеҝғзҡ„еӨҡйӣҶзҫӨй•ңеғҸ

еҸ‘иЎЁдәҺ 2020-02-14 жӣҙж–°дәҺ 2026-07-05 еҲҶзұ»дәҺ Message Queue - Kafka жң¬ж–Үеӯ—ж•°пјҡ 1k йҳ…иҜ»ж—¶й•ҝ ≈ 1 еҲҶй’ҹ

и°ғз ”з»“жһңе’Ңе»әи®®

й’ҲеҜ№жҳҜеҗҰеҸҜд»Ҙи·Ёж•°жҚ®дёӯеҝғйғЁзҪІKafkaйӣҶзҫӨзҡ„й—®йўҳпјҢзӯ”жЎҲжҳҜеҸҜиЎҢпјҢдҪҶжҳҜдёҚе»әи®®гҖӮе»әи®®е…Ҳж”ҜжҢҒеҚ•йӣҶзҫӨйғЁзҪІпјҢд»Ҙз®ҖеҢ–иҝҗз»ҙжҲҗжң¬пјҢж”ҜжҢҒеӨҡең°еӨҡеә”з”ЁдёҡеҠЎдёҠзәҝгҖӮ

еҸҜиЎҢжҖ§дёҺи·ЁзҪ‘з»ңйғЁзҪІдә§з”ҹзҡ„й—®йўҳ

Kafkaе®һйҷ…дёҠе№¶дёҚзҰҒжӯўе°ҶBrokerйғЁзҪІеҲ°дёҚеҗҢзҡ„зҪ‘з»ңең°еҹҹгҖӮдҪҶжҳҜпјҢKafkaзҡ„и®ҫи®ЎжҳҜжҢүз…§еҚ•дёӘж•°жҚ®дёӯеҝғиҝӣиЎҢи®ҫи®ЎгҖҒејҖеҸ‘гҖҒжөӢиҜ•е’Ңи°ғдјҳзҡ„гҖӮд»ҺKafkaзҡ„е®ҳж–№ж–ҮжЎЈ[1]жқҘзңӢпјҢе№¶дёҚжҺЁиҚҗе°ҶйӣҶзҫӨи·ЁзҪ‘з»ңйғЁзҪІгҖӮ з”ұдәҺBrokerеҲҶеёғеңЁдёҚеҗҢзҡ„и·Ёең°еҹҹзҡ„зҪ‘з»ңпјҲWANпјү, йҷӨдәҶж•°жҚ®еҗҢжӯҘдјҡжһҒеӨ§ең°еўһеҠ е»¶иҝҹд№ӢеӨ–пјҢ е№¶дё”еҰӮжһңз”ұдәҺзҪ‘з»ңдёӯж–ӯж— жі•дҝқиҜҒzkе§Ӣз»ҲеҸҜз”ЁпјҢеёҰжқҘе…¶д»–й—®йўҳпјҢжҜ”еҰӮпјҡ

  1. Partition Leaderж— жі•йҖүдёҫгҖӮжҲ–иҖ…з”ұдәҺзҪ‘з»ңй—®йўҳпјҢжҹҗдёӘBrokerйў‘з№ҒжҺүзәҝеҜјиҮҙйӣҶзҫӨдёҚзЁіе®ҡгҖӮ
  2. еңЁProducerжҺЁйҖҒж•°жҚ®ж—¶еҖҷпјҢдә§з”ҹе»¶иҝҹпјҲжҺЁйҖҒеҲ°remote BrokerпјүгҖӮжҲ–иҖ…з”ұдәҺзҪ‘з»ңй—®йўҳпјҢж— жі•иҺ·еҸ–ACKгҖӮ

и·Ёе№ҝеҹҹзҪ‘йғЁзҪІStretch Cluster

  2. жңүйңҖиҰҒејӮең°е®№зҒҫе’Ң100%жҒўеӨҚзҡ„йңҖжұӮ
  3. й…ҚзҪ®еҸӮж•°min.isrе’Ңacks=allгҖӮиҝҷж ·е°ұеҸҜд»ҘиҰҒжұӮејӮең°жүҖжңүзҡ„BrokerйғҪеҸҜд»Ҙж”¶еҲ°пјҢе№¶дё”зЎ®и®Өж¶ҲжҒҜгҖӮ
